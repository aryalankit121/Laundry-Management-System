# 🧺 Laundry Management System

A console-based Python application designed to help laundromat owners calculate customer charges, manage pricing, and track customer spending history. This project was developed while learning Python fundamentals and demonstrates the use of functions, dictionaries, loops, and file-like data management in a menu-driven application.

---

## ✨ Features

- 🧺 Calculate laundry cost based on weight
- 💵 Automatic subtotal, tax, and total calculation
- 👤 Store and track customer spending using unique User IDs
- 📊 View spending history for individual customers
- 📋 Display the complete customer database
- ⚙️ Update price per pound
- 🏷️ Update tax rate
- ✔️ Confirmation prompts to prevent accidental changes
- 📜 Menu-driven interface for easy navigation

## 🎮 Program Menu

After launching the application, you will be presented with four options:

```
1. Calculate total for a customer
2. Change price per pound and tax rate
3. View customer spending history
4. Quit
```

---

## 📋 Features Overview

### 🧺 Calculate Customer Total

- Enter a customer's unique User ID.
- Enter the laundry weight in pounds.
- The program calculates:
  - Subtotal
  - Tax
  - Final Total
- Customer spending is automatically added to their cumulative history.

---

### ⚙️ Manage Pricing

The owner can update:

- Price per pound
- Tax rate
- Both values simultaneously

Changes require confirmation before being applied.

---

### 📊 Customer Spending History

View:

- A specific customer's total spending
- The complete customer database
